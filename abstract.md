# Abstract

## Predictive Analytics for Student Academic Performance in Nigerian Tertiary Institutions Using Machine Learning Techniques

Academic performance in tertiary institutions serves as a critical indicator of educational effectiveness and student success, directly influencing graduation rates, employability prospects, and national human capital development. In Nigerian tertiary institutions, traditional approaches to monitoring student performance remain largely reactive, with interventions typically implemented only after academic failures have occurred. This study addresses the pressing need for proactive, data-driven approaches to identify at-risk students and enable timely academic support through the development of an interpretable machine learning-based web application.

The research employs a comprehensive methodology that integrates advanced machine learning techniques with explainable artificial intelligence (XAI) frameworks to create a predictive system tailored to the Nigerian educational context. Using historical student data encompassing academic records, demographic characteristics, and behavioral patterns, this study develops and evaluates supervised machine learning models for academic performance prediction. The methodology incorporates Synthetic Minority Over-sampling Technique (SMOTE) to address class imbalances in the dataset, ensuring fair and accurate model performance across different student populations.

Two primary machine learning algorithms, Random Forest and LightGBM, are implemented and comparatively evaluated to determine their effectiveness in predicting student academic outcomes. The Random Forest model provides robust ensemble learning with strong interpretability, while LightGBM offers gradient boosting capabilities with superior computational efficiency. Both models are trained on preprocessed student data features including cumulative grade point averages, course failure rates, attendance patterns, library usage statistics, and socioeconomic indicators.

A key innovation of this research is the integration of SHAP (SHapley Additive exPlanations) as an explainable AI layer to enhance model transparency and interpretability. The SHAP framework provides detailed explanations for individual predictions, enabling educators and administrators to understand the specific factors contributing to each student's risk assessment. This interpretability component is crucial for building trust in the predictive system and facilitating informed decision-making by non-technical stakeholders.

The system architecture combines Python-based machine learning backend services with a Node.js web application framework, creating an integrated platform for real-time prediction and visualization. The web-based dashboard provides intuitive interfaces for academic advisors to input student data, receive risk assessments, and access detailed explanations of prediction outcomes. The system implements secure authentication mechanisms and comprehensive data validation to ensure privacy and data integrity.

Preliminary results demonstrate that both Random Forest and LightGBM models achieve high predictive accuracy in identifying students at risk of academic failure, with the LightGBM model showing slightly superior performance in terms of precision and recall metrics. The SHAP-based explanations reveal key factors influencing academic performance, including previous semester GPA, course failure rates, attendance patterns, and socioeconomic background. These insights enable targeted interventions and personalized academic support strategies.

The significance of this research extends beyond technical implementation to address critical challenges in Nigerian higher education. By providing early warning capabilities and interpretable insights, the system empowers educational institutions to move from reactive to proactive academic management. The web-based platform ensures accessibility for educators without technical expertise, bridging the gap between advanced analytics and practical educational applications.

This study contributes to the growing body of knowledge in educational data mining while addressing the specific context of Nigerian tertiary institutions. The developed system demonstrates the practical applicability of machine learning techniques in real-world educational settings and provides a framework for similar implementations across diverse educational environments. The research findings support evidence-based educational policies and highlight the potential of predictive analytics to transform academic support services in Nigerian higher education.

The successful implementation of this predictive analytics platform represents a significant step toward data-driven decision-making in Nigerian tertiary institutions, offering a scalable solution for improving student outcomes through early identification of academic risks and timely intervention strategies.
